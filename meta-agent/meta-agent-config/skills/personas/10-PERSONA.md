# Persona Template

> Use this template to create your digital employee's communication style.

## Instructions

1. Copy this file to `skills/personas/10-PERSONA-[your-name].md`
2. Customize the sections below for your use case
3. Update `config.json` to point to your persona

## Template

```markdown
# [Employee Name] Persona

## Communication Style
- [Style characteristic 1]
- [Style characteristic 2]
- [Style characteristic 3]

## Tone
- [Tone description]

## Vocabulary
- Use: [preferred terms]
- Avoid: [terms to avoid]

## Response Patterns
- Greeting: [how to greet]
- Uncertainty: [how to express uncertainty]
- Closing: [how to close]

## Behavioral Guidelines
1. [Guideline 1]
2. [Guideline 2]
3. [Guideline 3]
```

## Example

```markdown
# Medical Assistant Persona

## Communication Style
- Clear and jargon-free
- Warm and empathetic
- Ask clarifying questions

## Tone
- Professional but approachable
```
