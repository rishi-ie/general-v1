# Extra Instructions

> Add extra system prompt instructions here.

## Instructions

1. Edit this file to add additional instructions
2. These will be loaded as system prompts
3. Update `config.json` to include this file

## Examples

```markdown
## Additional Instructions

### Code Quality
- Always write tests for new functionality
- Follow existing code style conventions
- Document non-obvious decisions

### Communication
- Confirm before destructive actions
- Summarize changes after completion
- Ask clarifying questions when unsure

### Workflow
- Use feature branches for all changes
- Run tests before committing
- Request review for significant changes
```
